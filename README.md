# 📚 学習ログ（LOG ディレクトリ）

このディレクトリは、学習の記録や振り返りを年ごとに整理したものです。

## 📁 構成概要

- `2024/`：2024年の学習記録（例：CSS、HTML、Javascriptの学習など）
- `2025/`：2025年の学習記録（例：Next.jsの基礎、ポートフォリオの作成、LeetCode演習など）

## 📝 利用目的

- 自身の成長記録として、学習内容・成果・気づきを残す
- 後から見返せるように体系的に整理

## 💡 活用方法

- `learning-log.md` にログを記録
- `.md` ファイルに教材リンクや振り返りコメントを記入
- 必要に応じて、随時ファイルを追加
